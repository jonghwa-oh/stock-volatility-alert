## ❓ FAQ

### Q1. 라즈베리파이 없이 할 수 없나요?
**A:** 가능하지만:
- 컴퓨터를 24시간 켜야 함
- 전력 소비 높음 (50-200W)
- 소음 발생

### Q2. 클라우드는 안 되나요?
**A:** 가능하지만:
- 월 비용 높음 (~13,000원)
- 설정 복잡함
- 라즈베리파이가 더 경제적

### Q3. 종목을 20개 이상 하면?
**A:** 가능하지만:
- 분석 시간 증가
- API 호출 증가
- 텔레그램 메시지 너무 많음
- 20개 이하 권장

### Q4. 5분보다 짧게 체크하면?
**A:** 가능하지만:
- API 제한 주의
- 불필요한 알림 증가
- 5분이 적절함
