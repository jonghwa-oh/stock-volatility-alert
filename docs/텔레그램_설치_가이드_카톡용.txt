안녕하세요! 😊
매일 아침 주식 매수 전략을 텔레그램으로 받으실 수 있어요.
아래 순서대로 따라해주세요!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔹 1단계: 텔레그램 앱 설치

▫️ 아이폰: 앱스토어에서 "Telegram" 검색 후 설치
▫️ 안드로이드: 플레이스토어에서 "Telegram" 검색 후 설치

💡 파란색 종이비행기 아이콘이 텔레그램이에요!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔹 2단계: 텔레그램 가입

1. 앱을 열고 휴대폰 번호 입력
2. 문자로 받은 인증번호 입력
3. 이름 입력 (나중에 변경 가능)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔹 3단계: 주식 알림 봇 추가

1. 텔레그램 앱 하단의 🔍 검색 아이콘 클릭

2. 검색창에 다음을 입력:
   @stock_alert_sigma_bot

3. "Stock Alert Sigma Bot" 클릭

4. 하단의 "시작" 또는 "Start" 버튼 클릭

5. 봇이 인사 메시지를 보내면 성공! ✅

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔹 4단계: Chat ID 확인

1. 봇 채팅창에서 다음 메시지를 입력:
   /start

2. 봇이 답장으로 숫자를 알려줄 거예요
   (예: "Your Chat ID: 1234567890")

3. 이 숫자를 남편에게 카톡으로 전달해주세요!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 완료!

Chat ID를 전달해주시면
매일 아침 8:50에 주식 매수 전략이
차트와 함께 자동으로 전송됩니다! 📊

관심 종목과 투자 금액은
남편이 설정해드릴게요! 💰

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❓ 문제가 있으면?

▫️ 봇을 찾을 수 없어요
  → @stock_alert_sigma_bot 정확히 입력했는지 확인
  → 앞에 @ 기호도 꼭 포함!

▫️ 봇이 답장을 안 해요
  → "시작" 버튼을 눌렀는지 확인
  → /start 명령어를 입력해보세요
