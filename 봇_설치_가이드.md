## 2단계: 봇과 대화 시작 🤖

### 방법 1: 직접 검색 (추천!)

1. 텔레그램 앱 실행
2. 상단 **검색창** 클릭
3. 다음을 정확히 입력:
   ```
   @stock_alert_sigma_bot
   ```
4. 봇이 나타나면 클릭
5. 하단의 **"시작"** 또는 **"START"** 버튼 클릭
6. 다음 메시지 입력:
   ```
   /start
   ```

## 📱 봇 사용 방법

### 자동 알림 받기

매일 자동으로 다음 정보를 받게 됩니다:

✅ **일일 변동성 분석 리포트**
- 6개 종목 분석 결과
- 각 종목별 차트 이미지
- 매수 목표가 정보

### 알림 예시

**1. 시작 메시지**
```
📊 일일 변동성 분석 시작
══════════════════════════
⏰ 2024-11-29 15:30
📈 분석 종목: 6개
```

**2. 개별 종목 분석**
```
📊 코스피200
══════════════════════════

💰 현재가: 554.00
📈 평균 일일 변동: +0.23%
📊 표준편차: 1.47%
   (하루 평균 ±1.47% 움직임)

🎯 매수 목표가:
   📍 1차: 545.84
      (하루 1.47% 하락 시)
   📍 2차: 537.67
      (하루 2.95% 하락 시)
```

**3. 차트 이미지**
📊 각 종목마다 3개 그래프 포함
- 가격 차트
- 일일 변동률 시계열
- 변동률 분포

**4. 전체 요약**
```
📊 일일 변동성 분석 리포트
═══════════════════════════

🎯 변동성 순위:

1. SOXL
   • 표준편차: 7.46%
   • 현재가: 41.26

2. TQQQ
   • 표준편차: 4.49%
   • 현재가: 54.54
...
```

## ❓ 문제 해결

### Q1. 봇을 찾을 수 없어요
**A:** 다음을 확인하세요:
- `@stock_alert_sigma_bot` 정확히 입력했는지
- `@` 기호 포함했는지
- 텔레그램 앱이 최신 버전인지

### Q2. 시작 버튼이 안 보여요
**A:** 
- 이미 시작한 상태일 수 있습니다
- `/start`를 직접 입력해보세요
- 봇을 차단한 적이 있다면 차단 해제 필요

### Q3. 메시지가 안 와요
**A:**
1. 텔레그램 알림 설정 확인
2. 봇을 차단하지 않았는지 확인
3. `/start`를 다시 입력해보세요

### Q4. 봇이 응답하지 않아요
**A:**
- 관리자에게 봇 서버 상태 확인 요청
- 잠시 후 다시 시도
